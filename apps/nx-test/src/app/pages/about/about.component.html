Over deze applicatie

Deze applicatie is gemaakt in het kader van het vak 'Clientside web frameworks' van Avans Hogeschool Breda.

Course It

Op deze applicatie kunnen studenten zich inschrijven voor cursussen. Deze cursussen worden aangeboden door docenten.
Studenten en docenten kunnen zich inschrijven voor een cursus.
Studenten mogen geen workshops aanmaken maar kunnen wel workshops volgen.
Docenten mogen wel workshops aanmaken en kunnen deze ook volgen.
Een admin kan alles doen, en is de enige die subjects aan kan maken, docent moeten dus naar een admin als zei een nieuwe subject willen.
Om een workshop te volgen moet deze gekocht worden met credits, deze credits kunnen worden gekocht met een voucher of betaalpas.
Een workshop kan gratis zijn, en hoeft dan dus niet gekocht te worden maar een gebruiker moet altijd inloggen om de workshop te volgen
Wanneer een workshop gekocht is blijft deze voor altijd beschikbaar.

Userstories

Nr Userstory Evt toelichting
1 Als gebruiker wil ik een overzicht van workshops kunnen zien, zodat ik kan zien welke workshops beschikbaar zijn. Overzicht is een grid.
2 Als ingelogde docent wil ik zelf workshops kunnen aanmaken, zodat ik deze aan gebruikers aan kan bieden.
3 Als ingelogde gebruiker wil ik een workshop kunnen kopen, zodat ik deze kan volgen.
4 Als ingelogde admin wil ik alle data kunnen veranderen, zodat ik de website kan beheren.
5 Als ingelogde docent wil ik mijn workshops kunnen aanpassen, zodat ik deze kan updaten. Dit mag alleen bij workshops aangemaakt door deze docent.
6 Als ingelogde gebruiker wil ik mijn profiel kunnen aanpassen, zodat ik mijn gegevens kan updaten.
7 Als ingelogde gebruiker mag ik geen toegang hebben tot data die ik niet aan mag passen, en wil ik dit niet zien in de navigatiebalk.
